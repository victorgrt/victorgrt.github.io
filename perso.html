Le jeu de la Vie de Conway

Le projet libft de 42Born2Code consiste à recoder un ensemble de fonctions de la bibliothèque standard C, telles que celles pour manipuler des chaînes de caractères, des tableaux, des listes chaînées, etc. L'objectif est de créer une bibliothèque personnelle, nommée "libft", qui pourra être utilisée dans les projets futurs à 42. Le projet permet également de se familiariser avec la gestion de la mémoire, les pointeurs, les structures de données et les outils de compilation.

TLDR : Librairie de fonctions, Liste chaînées, Makefile, C

Blackjack-21

Le projet get_next_line de 42Born2Code consiste à créer une fonction en C qui permet de lire un fichier ligne par ligne à partir d'un file descriptor. L'objectif est d'implémenter une fonction nommée "get_next_line" qui peut être appelée plusieurs fois pour obtenir les lignes successives d'un fichier sans avoir à tout lire en une seule fois. Pour cela, la fonction utilise la fonction "read()" pour lire le fichier, ainsi que la fonction "open()" pour ouvrir le file descriptor correspondant. Le projet permet de se familiariser avec les appels système Unix, la gestion de la mémoire, les pointeurs, les tableaux et les boucles en C.

TLDR : Lecture de fichier, read(), File Director, Ligne par ligne

Hangman

Le projet Born2beroot consiste à installer et configurer un serveur Debian minimal dans une machine virtuelle. L'objectif est de sécuriser le serveur en configurant un pare-feu, un accès SSH sécurisé, des règles de journalisation, et en appliquant les mises à jour de sécurité. Le projet permet également de se familiariser avec les commandes Linux et la gestion de la sécurité des serveurs.

TLDR : Machine Virtuelle, Configuration Mémoire, Permission, Scripts d'entretien, Passwd Policy
