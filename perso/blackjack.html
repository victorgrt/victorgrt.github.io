Le jeu du Blackjack

Le jeu de blackjack, également connu sous le nom de 21, est l'un des jeux de casino les plus populaires au monde. Les origines exactes du blackjack ne sont pas claires, mais il est généralement admis que le jeu est apparu pour la première fois en France au XVIIIe siècle. Le jeu a ensuite été introduit en Amérique du Nord, où il est rapidement devenu un favori des joueurs de casino. Les règles du blackjack sont simples: les joueurs essaient de battre le croupier en obtenant une main de cartes d'une valeur totale de 21, ou le plus proche possible de 21, sans dépasser. Les cartes numériques ont leur propre valeur, tandis que les cartes à face ont une valeur de 10 et les As peuvent avoir une valeur de 1 ou 11 selon la main. Le blackjack est un jeu passionnant et rapide qui repose sur une combinaison de chance et de stratégie.

Tester mon programme (Linux)

Le code source de mon implémentation se trouve sur Github, vous pouvez donc cloner ce projet en vous rendant sur mon Github ou alors avec la commande :

git clone https://github.com/victorgrt/blackjack-21

Une fois le dossier clonné, vous n'avez plus qu'à lancer la commande :

make && ./a.out

Cela va compiler le programme et lancer l'éxécutable dans le terminal. Vous pouvez maintenant parier puis soit demander une carte, vous arrêter là ou abandonner la partie. GLHF
