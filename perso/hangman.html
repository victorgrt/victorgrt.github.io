Le jeu du pendu

Le jeu du pendu est un jeu populaire qui a été créé il y a plusieurs siècles. L'origine exacte du jeu n'est pas claire, mais il est probable qu'il ait été inventé en Europe ou en Angleterre. Au fil des ans, le jeu a été adapté et modifié pour inclure des règles différentes et des variantes régionales, mais l'idée de base reste la même : deviner un mot en proposant des lettres jusqu'à ce que toutes les lettres soient trouvées ou jusqu'à ce que le joueur ait épuisé toutes ses chances.

Le joueur doit deviner un mot en proposant des lettres une par une. Si la lettre proposée est dans le mot, elle est placée dans sa position respective. Si la lettre n'est pas dans le mot, le joueur est pénalisé en ajoutant une partie de la silhouette d'un homme suspendu à une corde. Le but est de deviner le mot avant que la silhouette ne soit complétée, sinon le joueur perd. Le nombre de tentatives dépend des règles établies, mais généralement, les joueurs ont environ six chances pour deviner le mot.

Possibles améliorations :

  • Problême d'affichage
  • Ajout de nouveaux mots
  • Prendre en compte les caractères spéciaux
  • Fix les leaks

Tester mon programme (Linux)

Il vous suffira d'avoir le compilateur gcc installé pour compiler le programme dans un executable.

Le code source de mon implémentation se trouve sur Github, vous pouvez donc cloner ce projet en vous rendant sur mon Github ou alors avec la commande :

git clone github.com/victorgrt/hangman

Une fois le dossier clonné, utiliser la commande ci-dessous pour lancer la compilation et lancer l'éxécutable :

make && ./a.out

Une fois sur le programme lancé un mot ainsi que l'échaffau devraient apparaître. Vous pouvez essayer des lettres directement dans le terminal.
